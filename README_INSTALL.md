# 🛠️ Arvayon CRM - Local Installation Guide

To install and run Arvayon CRM on your local system as an application, follow these simple steps:

### 1️⃣ Prerequisites
Make sure you have **Docker Desktop** installed on your computer.
- [Download Docker Desktop](https://www.docker.com/products/docker-desktop/)

### 2️⃣ Installation
1. Download the project folder to your system.
2. Open the folder.
3. **Double-click** on the `START_CRM.bat` file.
   - This will automatically start the Database, Backend API, AI Service, and Frontend UI.

### 3️⃣ Running as a Desktop App (PWA)
Once the system is running in your browser (at `http://localhost:3000`):
1. Look at the Address Bar in **Google Chrome** or **Microsoft Edge**.
2. Click the **"Install App"** icon (or go to Menu > Save and Share > Install page as app).
3. This will create an **Arvayon CRM** icon on your Desktop and Taskbar.
4. You can now use it exactly like a standalone desktop application!

### 🛑 How to Stop
To close the system, simply close the `START_CRM.bat` command window or run `docker-compose down`.
